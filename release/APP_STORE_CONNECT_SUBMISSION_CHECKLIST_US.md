- [ ] Contact first name/last name/email/phone filled

- [ ] If prompted, answer export compliance truthfully based on current encryption usage.

## 9) Build and Submit

- [ ] Select the uploaded production build for this version
- [ ] Resolve all warnings in the version page
- [ ] Submit for Review
- [ ] Release option: `Manual Release` for first launch

## 10) Final Pre-Submit Quality Gate

- [ ] App opens cleanly on fresh install
- [ ] Settings > Data & Privacy page is visible and accurate
- [ ] Consent, export, and delete-all-data flows work end-to-end

- [ ] No references to EU release plans in metadata or review notes
