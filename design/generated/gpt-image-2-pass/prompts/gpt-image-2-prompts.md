# GPT Image 2 Prompt Specs

## Base App Icon
Use case: logo-brand
Asset type: iOS app icon master, 1024 x 1024
Primary request: refined Catholic Fasting app icon using the existing monstrance identity as the core mark.
Style/medium: premium vector-friendly sacred icon, restrained, crisp, App Store-ready.
Composition/framing: centered monstrance silhouette, strong small-size readability, clear negative space, rounded-square-safe composition.
Lighting/mood: quiet Eucharistic glow, calm and reverent, not theatrical.
Color palette: deep aubergine background, warm ivory and muted gold mark, subtle contrast.
Constraints: no text, no watermark, no people, no photo-real church interiors, keep the core monstrance silhouette recognizable.
Avoid: generic AI glow, busy rays, excessive gradients, thin details that disappear at 29 px, clip-art style.

## Seasonal Icon Family
Use case: logo-brand
Asset type: iOS alternate app icon masters, 1024 x 1024 each
Primary request: four seasonal variants of the same Catholic Fasting monstrance icon for Advent, Christmas, Lent, and Easter.
Style/medium: same premium vector-friendly icon system as the base icon.
Composition/framing: identical silhouette and geometry across all four variants.
Lighting/mood: Advent quiet violet anticipation; Christmas warm gold and ivory; Lent sober violet/ash; Easter luminous ivory/gold with restrained joy.
Constraints: no text, no watermark, no new central symbol, no major silhouette changes between seasons.
Avoid: holiday kitsch, neon, snowflakes as primary motif, Easter eggs, overly ornate rays, inconsistent mark weight.

## Sacred Hero Imagery
Use case: stylized-concept
Asset type: in-app wide sacred hero image
Primary request: reverent Catholic sacred hero art for fasting, prayer, abstinence, and liturgical rhythm.
Style/medium: editorial sacred illustration with painterly texture and modern restraint.
Composition/framing: wide crop, clear focal center, enough calm negative space for overlaid title text.
Lighting/mood: quiet chapel light, contemplative, premium, not dramatic.
Color palette: parchment, muted gold, deep plum, warm stone, soft candlelight.
Constraints: no text, no watermark, no distorted hands/faces, no doctrinally confusing symbols, no crowded altar scenes.
Avoid: fantasy glow, stock Catholic wallpaper, excessive incense haze, over-rendered realism.
