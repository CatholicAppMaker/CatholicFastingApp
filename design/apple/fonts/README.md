# Local Apple Font References

This folder contains practical local font references copied from the current macOS install.

## Included
- `SFNS.ttf`
- `SFNSItalic.ttf`
- `SFNSMono.ttf`
- `SFNSMonoItalic.ttf`
- `SFNSRounded.ttf`
- `NewYork.ttf`
- `NewYorkItalic.ttf`

## Notes
- `SFNS.ttf` is the closest practical local system reference to Apple's San Francisco / SF Pro UI family on this Mac.
- `NewYork.ttf` is Apple's serif family used for more editorial, literary, and formal presentation.
- The separate Apple Developer downloads for `SF Pro` and `New York` appear to be sign-in gated, so they were not automatically fetched from the web.
- These local copies are intended as design references for internal project work on `Catholic Fasting App`, especially for `4.0` design exploration.

## Recommended usage in this app
- Use `SFNS`-family fonts as the default UI baseline reference.
- Use `New York` sparingly for devotional headings, feast/season emphasis, or editorial pull-quote moments.
